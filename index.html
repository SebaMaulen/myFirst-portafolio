Mis Proyectos

Plataforma E-commerce

Desarrollé una plataforma de comercio electrónico completa que permite a los usuarios navegar por productos, añadirlos a un carrito de compras, y completar sus compras mediante una pasarela de pago segura.

Sistema de reserva Restaurantes

Desarrollé una aplicación web que permite a los usuarios registrar y seguir sus rutinas de ejercicio. La aplicación incluye gráficos para visualizar el progreso y un componente de gamificación para motivar a los usuarios.

Sistema gestion Inventario

Desarrollé un sistema para gestionar el inventario de una pequeña empresa. La aplicación permite el seguimiento de productos, la gestión de proveedores y la generación de informes sobre el estado del inventario.

Aplicacion de notas

Creé una aplicación sencilla de notas que permite a los usuarios crear, editar y eliminar notas. La aplicación guarda las notas localmente y tiene una interfaz minimalista y fácil de usar.
